This theme uses the visitor's operating-system UI font stack
(-apple-system, "Segoe UI", Roboto, Helvetica, Arial, sans-serif) instead of
a bundled webfont, to avoid extra HTTP requests and keep the theme fully
self-contained with zero external dependencies.

Drop any .woff2 files here and register them with @font-face in
assets/css/main.css if you'd like to use a custom typeface instead.
